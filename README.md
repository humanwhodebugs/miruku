# ミルク (Miruku)

**ミルク** is my personal dotfiles collection, built for a lightweight, keyboard-focused workflow on Arch Linux. It’s crafted specifically for my ThinkPad X230i, optimized for performance and simplicity.

## Features

- **Window Manager**: `bspwm` — A minimal tiling window manager
- **Status Bar**: `eww` — Fully customizable and styled with Catppuccin Latte
- **Terminal**: `kitty` — Fast and modern GPU-based terminal
- **Colorscheme**: Catppuccin Latte across all tools

## Design Goals

- Minimal, efficient, and distraction-free
- Easy to navigate entirely with the keyboard
- Modular and hackable — easy to change components

## Setup (Manual)

git clone https://github.com/humanwhodebugs/miruku ~/miruku
cd ~/miruku/Install

Run both install script.
