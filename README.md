# Miruku (ミルク)

_A dotfiles setup forged in simplicity, crafted with intention._

**Miruku (ミルク)** — meaning _"milk"_ in Japanese — is a minimalist, performance-focused dotfiles collection tailored for low-spec hardware without compromising on power, style, or flexibility. Inspired by the philosophy of doing more with less, Miruku is designed to offer a creamy and calm computing experience, squeezing every drop of efficiency from the system while maintaining visual clarity, elegance, and user control.

> No bloat. No distractions. Just you and the machine — in perfect sync.

---

## 🌱 Features

- ⚡ Lightweight: Sub-600MB RAM usage on idle
- 🔥 Optimized: ThinkFan, TLP, Thermald for cool, efficient performance
- 🎨 Beautifully Riced: EWW bar, consistent colors, and clean visual design
- 🐧 Arch Linux + ThinkPad focused setup
- 🌈 Catppuccin Latte colorscheme throughout

---

## 🧠 Philosophy

- **Simplicity is power** — Every config has a purpose
- **Transparency over automation** — Know your system, don’t hide from it
- **Beauty meets functionality** — Minimal doesn’t mean boring
- **Performance-first** — Even low-spec setups can feel snappy when done right
