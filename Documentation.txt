Arch Linux Installation and Configuration Guide

INSTALLATION PROCESS

1. Connect to Internet
iwctl
station wlan0 connect [WLAN_NAME]

After connecting, type `clear` and then `exit`.

2. Update System Clock
Test connection first, then type:
timedatectl

If synchronized, set keymap when installing Arch:
loadkeys jp106

3. Disk Partitioning
mkfs.ext4 /dev/sda
fdisk /dev/sda

Create partitions as follows:
- 60GB for root
- 8GB for swap
- Rest for home

Format partitions:
mkfs.ext4 /dev/sda1
mkswap /dev/sda2
swapon /dev/sda2
mkfs.ext4 /dev/sda3

Mount partitions:
mount /dev/sda1 /mnt
mkdir /mnt/home
mount /dev/sda3 /mnt/home

4. Install Essential Packages
pacstrap -K /mnt base linux-firmware linux-lts (or linux-zen) base-devel git grub sudo vim intel-ucode thermald tlp networkmanager

5. Generate Fstab
genfstab -U /mnt >> /mnt/etc/fstab

6. Chroot
arch-chroot /mnt

7. Set Time Zone
ln -sf /usr/share/zoneinfo/Asia/Jakarta /etc/localtime
hwclock --systohc

8. Localization
Edit `/etc/locale.gen` and uncomment `en_US.UTF-8 UTF-8`, then generate locales:
locale-gen

Create `/etc/locale.conf`:
LANG=en_US.UTF-8

Set keyboard layout in `/etc/vconsole.conf`:
KEYMAP=jp106

9. Network Configuration
Create hostname file `/etc/hostname`:
archlinux

10. Root Password and User Creation
Set root password:
passwd

Create new user:
useradd -m -G wheel -s /bin/bash rei
passwd rei

11. Enable sudo for Non-Root User
EDITOR=vim visudo

Search for `%wheel` and uncomment:
%wheel ALL=(ALL:ALL) ALL

12. Enable 32-bit Support
Edit `/etc/pacman.conf` and uncomment:
[multilib]
Include= /etc/pacman.d/mirrorlist

13. Install Bootloader
grub-install /dev/sda
grub-mkconfig -o /boot/grub/grub.cfg

14. Enable Services
systemctl enable thermald
systemctl enable tlp
systemctl enable NetworkManager

15. Complete Installation
exit
umount -R /mnt
reboot

POST-INSTALLATION CONFIGURATION

1. Configuring ArchLinuxCN
Activate ArchLinuxCN Repo
sudo vim /etc/pacman.conf

Add to end of file:
[archlinuxcn]
Server = https://repo.archlinuxcn.org/$arch

Sync repo and install keyring:
sudo pacman -Sy
sudo pacman -S archlinuxcn-keyring

2. Installing Paru
Install rustup:
sudo pacman -S rustup
rustup install stable
rustup default stable

Install paru (AUR Helper)
git clone https://aur.archlinux.org/paru.git
cd paru
makepkg -si

3. Configuring BSPWM Double Border
BSPWM Dependencies
git clone https://github.com/wmutils/opt
cd opt
make
sudo make install

4. Configuring ThinkFan
Install ThinkFan and lm_sensors (can skip lm_sensors if already installed):
paru -S thinkfan lm_sensors  # install

Run this in terminal:
echo "options thinkpad_acpi fan_control=1" | sudo tee /etc/modprobe.d/thinkfan.conf
sudo sh -c 'echo "enable" > /proc/acpi/ibm/fan'      # manual enable fan mode on setup
sudo modprobe thinkpad_acpi fan_control=1            # load module with manual fan control option
sudo mkinitcpio -P                                   # regenerate initramfs

Configure ThinkFan:
sudo vim /etc/thinkfan.conf

sensors:
  - hwmon: /sys/class/hwmon
    name: coretemp
    indices: [1, 2, 3]  # Package, Core 0, Core 1

  - hwmon: /sys/class/hwmon
    name: thinkpad
    indices: [1]  # CPU temperature from thinkpad sensor

fans:
  - tpacpi: /proc/acpi/ibm/fan

levels:
  - [0, 0, 50]       # Fan off until 50°C
  - [1, 45, 55]      # Level 1: Silent mode
  - [2, 50, 60]      # Level 2: Quiet fan
  - [3, 55, 65]      # Level 3: Moderate fan
  - [4, 60, 70]      # Level 4: Medium fan
  - [5, 65, 75]      # Level 5: High fan
  - [7, 70, 85]      # Level 7: Maximum fan speed

When finished, run this:
sudo systemctl enable thinkfan                        # enable service so it will run automatically after startup
sudo systemctl start thinkfan                         # start service
sudo journalctl -xeu thinkfan.service                 # check service logs
sudo reboot                                           # reboot so every configuration and kernel reboot automatically runs

Do this after reboot to check if thinkfan is enabled:
❯ cat /proc/acpi/ibm/fan
status:		enabled
speed:		3779
level:		3
commands:	level <level> (<level> is 0-7, auto, disengaged, full-speed)
commands:	enable, disable
commands:	watchdog <timeout> (<timeout> is 0 (off), 1-120 (seconds))

8. TLP Power Management Setup
Edit `/etc/tlp.conf`:
vim /etc/tlp.conf

Uncomment:
CPU_SCALING_GOVERNOR_ON_BAT=powersave
CPU_SCALING_GOVERNOR_ON_AC=performance

Restart service:
systemctl restart tlp

9. Kernel Configuration
Edit GRUB defaults:
sudo nano /etc/default/grub

Update kernel parameters:
GRUB_CMDLINE_LINUX_DEFAULT="loglevel=3 quiet rd.system.show_status=auto rd.udev.log_priority=3 mitigations=off noibrs noibpb nopti nospectre_v2 l1tf=off zswap.enabled=1 zswap.compressor=lz4 zswap.max_pool_percent=20 elevator=bfq io_uring=1"

Update GRUB and reboot:
sudo grub-mkconfig -o /boot/grub/grub.cfg
reboot

11. Git Repository Cloning
git clone git@github.com:humanwhodebugs/repo-name.git

12. System Boot Optimization

Check boot time:
systemd-analyze blame

Optimize journal:
sudo vim /etc/systemd/journald.conf

Edit to:
Storage=volatile

Restart journal:
sudo systemctl restart systemd-journald

Optimize mount options in `/etc/fstab`:
sudo vim /etc/fstab

Change:
UUID=xxxxx  /      ext4  rw,realtime  0 1
UUID=xxxxx  /home  ext4  rw,realtime  0 2

To:
UUID=xxxxx  /      ext4  rw,noatime  0 1
UUID=xxxxx  /home  ext4  rw,noatime  0 2

Reboot to apply changes.
